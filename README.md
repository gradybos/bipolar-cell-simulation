## Abstract
Biological nervous systems have evolved over millions of years, and are highly efficient. With artificial neural networks exploding in popularity, it is important to remember the biology in which they are rooted. Bipolar cells perform natural edge detection in the retina, so the performance of a model bipolar cell can be compared to other traditional edge detection methods, such as a Laplacian filter. The bipolar cell is inherently 3x more efficient than the Laplacian filter, while performing marginally better given Pratt's Figure of Merit as a metric.

## Installation
To run the code, create a virtual environment in which to store the packages using the following command:

python -m venv .venv

To install the packages, run the following command:

pip install -r requirements.txt

The dataset will download automatically to the project directory upon running all the cells.
